2. User Guide

Flip the ‘Power Switch’ to power on/off the device
Select the features on the OLED by pressing the ‘Up/Down button’. The features are:
  • ‘Read ID’: read ID card
  • ‘Simulate ID’: emulate/spoof as an ID card
  • ‘Write ID’: write ID to blank card
  • ‘Update’: update the firmware for the device
